Comandos gerais

Comandos sobre imagem, contêiner, dockerfile e docker-compose, além de significado de atributos e parâmetros estarão aqui!

Listar imagens
docker images
Baixar imagens
docker pull <imagem>
Exemplo instalando Ubuntu: docker pull ubuntu
Remover imagem
docker rmi <imagem_id_ou_nome>
Exemplo removendo Ubuntu: docker rmi ubuntu
Histórico de execuções
docker container ls -a
Lista de execuções
docker ps
Informações sobre um contêiner em execução
docker inspect
Executar comandos em um contêiner em execução
docker exec <nomeContêiner> bash
O comando docker exec -it caixa bash abre um ambiente bash para manuseamento do contêiner "caixa"
Rodar imagens
docker run
Listar as portas de um contêiner
docker port <idDoContêiner>
docker port 8czeb3274
Mapear automaticamente as portas de um contêiner
-P
Mapear manualmente as portas de um contêiner
-p
docker run -d -p 8080:80 dockersamples/static-site
Rodar um docker-compose
docker-compose up;
Status dos contêineres de um docker-compose em execução
docker-compose ps;
Rodar um único serviço do docker-compose
docker-compose run service_name
Parâmetro para interagir com o terminal (junção de -i + -t)
-it
docker run -it ubuntu bash
Ambiente da interação
bash
docker run -it ubuntu bash
Logar no Dockerhub com terminal
docker login docker.io

2. Dockerfile

O que é Dockefile?

Parabéns por chegar ao 3° assunto do GoDocker! Agora iremos explorar o assunto de Dockerfile.

Basicamente, os Dockerfiles, são scripts com uma série de comandos para criação de uma imagem . Neles, podemos  fazer uma série de coisas como executar comandos sh, criar variáveis de ambiente, copiar arquivos e pastas do host para dentro da imagem

Isso permite que você possa automatizar a criação de imagens para os contêineres da sua aplicação se basearem , uma vez que uma aplicação pode fazer o uso de vários Dockerfiles!
